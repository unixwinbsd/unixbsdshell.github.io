Swoole ist ein Framework für PHP, das auf asynchronen Co-Routinen basiert. Swoole wurde speziell für den Aufbau paralleler Systeme im großen Maßstab entwickelt. Swoole ist in der Programmiersprache C oder C++ geschrieben und wird als PHP-Erweiterung installiert. Wenn Sie Swoole PHP auf einem lokalen Computer wie FreeBSD erstellen möchten, müssen Sie es über pecl installieren. In der neuesten Version von FreeBSD ist das Swoole-Repository verfügbar, Sie können es ganz einfach direkt installieren und konfigurieren.

Swoole PHP stammt aus dem benachbarten China und wurde hauptsächlich von chinesischen Entwicklern entwickelt, die an groß angelegten Anwendungen für den chinesischen Markt arbeiten. Daher wurde dieses Produkt in stark frequentierten Produktionsumgebungen getestet und hat sich bewährt. Swoole ist eine App, auf die Sie sich bei Ihrer Arbeit wirklich verlassen können und die Spaß macht.

## A. Vorteile von Swoole PHP
Im Vergleich zu anderen Frameworks oder Software für asynchrone Programmierung wie Apache, Nginx und Node.js unterstützt Swoole die asynchrone Programmierung stärker durch Multithread-E/A-Module (HTTP-Server, WebSockets, TaskWorkers). Darüber hinaus unterstützt Swoole PHP auch die Protokolle Redis, CURL, TCP/UDP und Unix-Sockets.

Mit den Vorteilen von Swoole können Sie kostenlos Anwendungen mit PHP im großen Maßstab (Webserver) erstellen, Anwendungen mit API erstellen, Chatsysteme und CMS-Plattformen erstellen und sogar Anwendungen mit Echtzeit-Webdiensten erstellen.

Swoole bietet eine Reihe von Vorteilen, die andere Anwendungen wie NodeJS nicht haben, darunter mehrere Web-Worker und separate Task-Worker, Coroutine-Unterstützung und die Möglichkeit, Anforderungslimits deutlich zu erhöhen. Hier sind einige der Vorteile von Swoole PHP:
* event-driven
* full asynchronous non-blocking
* multi-thread reactor
* Supports Separate Task Workers
* millisecond timer
* Supports Multiple Web Workers
* async MySQL
* async task
* async read/write file system
* Can Increase Request Ceiling
* async dns lookup
* No Web Server Needed
* support IPv4/IPv6/UnixSocket/TCP/UDP
* Coroutine Support

Auf den ersten Blick ähneln die Funktionen von Swoole denen von NodeJS. Beide verfügen über Event Loop Funktionen und bieten asynchrones HTTP, Netzwerkclients und Sockets. Mit Swoole können Sie Netzwerkserver und vieles mehr erstellen.

## 2. Installieren Sie Swoole
Unter FreeBSD ist die Installation von Swoole sehr einfach, keine Konfigurationsdateien, rc.d muss nicht ausgeführt werden. Um es auszuführen, müssen Sie nur ein PHP Skript hinzufügen. Der folgende Befehl ist eine Anleitung zur Installation von Swoole unter FreeBSD.

```
root@ns3:~ # cd /usr/ports/devel/pecl-swoole
root@ns3:/usr/ports/devel/pecl-swoole # make config
```

Wenn Sie den Befehl **"make config“** ausführen, wird ein Menü angezeigt, das Sie zusammen mit Swoole aktivieren müssen. Sie sollten diese Optionen an die Anforderungen Ihres FreeBSD Systems anpassen.

Fahren Sie anschließend mit dem Befehl "make install clean" fort.

```
root@ns3:/usr/ports/devel/pecl-swoole # make install clean
```

root@ns3:/var # cd freebsd-swoole-app

Erstellen Sie eine Datei mit dem Namen "server.php".

```
root@ns3:/var/freebsd-swoole-app # touch server.php
root@ns3:/var/freebsd-swoole-app # chmod -R 775 server.php
```

Geben Sie in die Datei "/var/freebsd-swoole-app/server.php" das folgende Skript ein.

```
#!/usr/bin/php
<?php
$server = new \Swoole\Http\Server('192.168.5.2', 3000);
$server->on('Request', function (\Swoole\Http\Request $request, \Swoole\Http\Response $response) {
    $response->write(file_get_contents('index.html'));
    $response->status(200);
    $response->end();
});
$server->start();
```

Beachten Sie das Skript "\Swoole\Http\Server('192.168.5.2', 3000)", in diesem Skript läuft Swoole mit
IP = 192.168.5.2 (die private IP Ihres FreeBSD-Servers)
Port = 3000

Der nächste Schritt besteht darin, die Datei "server.php" auszuführen.

```
root@ns3:/var/freebsd-swoole-app # php server.php
```

Um die Ergebnisse anzuzeigen, öffnen Sie „Google Chrome“ und geben den Befehl "http://192.168.5.2:3000/" ein.

Als zusätzliches Lernmaterial erstellen Sie eine Datei mit dem Namen "app.php" und geben das folgende Skript in diese Datei ein.

```
root@ns3:/var/freebsd-swoole-app # touch app.php
root@ns3:/var/freebsd-swoole-app # chmod 775 app.php
root@ns3:/var/freebsd-swoole-app # ee app.php

<?php
$http = new swoole\Http\Server('192.168.5.2', 3000);
$http->on('Request', function ($request, $response) {
    $response->header('Content-Type', 'text/html; charset=utf-8');
    $response->end('<h1>Hello FreeBSD Swoole' . rand(1, 9999) );
});

$http->start();
?>
```

Im letzten Schritt führen wir die Datei "app.php" aus.

```
root@ns3:/var/freebsd-swoole-app # php app.php
```

Öffnen Sie „Google Chrome“, geben Sie „http://192.168.5.2:3000/“ ein und sehen Sie sich die Ergebnisse an. Die Worte "Hello FreeBSD Swoole" werden angezeigt.
