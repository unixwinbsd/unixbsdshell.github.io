To become a reliable data scientist, Jupyter is an important application to master. Data scientists are tasked with exploring data and turning problems into benefits. Therefore, tools like Jupyter can help. So, what is Jupyter and why is it so important?

Jupyter is an abbreviation of three programming languages, namely Julia (Ju), Python (Py), and R. These three programming languages ​​are important for a data scientist. Jupyter is a free web application used to create and share documents containing code, calculation results, visualizations, and text.

Jupyter functions to help you create computational narratives that explain the meaning of the data in it and provide insight into the data. In addition, Jupyter also facilitates collaboration between engineers and data scientists because of its ease in writing and sharing text and code.
